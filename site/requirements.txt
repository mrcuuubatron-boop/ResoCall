# Site-specific requirements and recommended frontend libraries

# Install tools:
# - Node.js (>=18)
# - pnpm (recommended)
#
# Install commands (Fedora example):
# sudo dnf install -y nodejs npm
# sudo npm install -g pnpm
#
# Install frontend deps (from package.json):
# Run inside the `site/` folder:
# pnpm install
