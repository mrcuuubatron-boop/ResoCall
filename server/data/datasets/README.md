# Sentiment Dataset

Date prepared: 2026-03-29

Files:
- `sentiment_dataset.csv`

## Scope
This is a project dataset artifact for training/evaluation experiments in sentiment module tasks.

## Balance
The dataset is class-balanced by design:
- positive: 10
- neutral: 10
- negative: 10

Total: 30 samples.

## Data protection
The current file contains synthetic text only and no personal data.
