# ResoCall
Проект по распознаванию и анализу звонков с помощью ИИ.

Краткое описание и цель проекта:
- Backend (FastAPI) в каталоге `server/` — API для загрузки и анализа аудио.
- Модуль ASR в `ASR/` — скрипты и зависимости для распознавания.
- Frontend (Next.js) в `site/` — веб-интерфейс.

Подробные инструкции по установке и запуску для фронтенда вынесены в `site/README.md`.
Инструкции по backend/ASR находятся в соответствующих каталогах и их `requirements.txt`.

## Быстрый запуск backend + site

Один скрипт запускает одновременно backend и frontend:

```bash
./run_all.sh
```

После запуска:
- backend: `http://127.0.0.1:8000`
- site: `http://127.0.0.1:3000`

Остановка обоих сервисов: `Ctrl+C`.
